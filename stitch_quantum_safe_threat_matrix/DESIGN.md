# Design System: Quantum IDS Dashboard
## Colors
- Primary Background: #0d1117
- Surface/Card Background: #161b22
- Accent Success: #7ee787 (Neon Green)
- Accent Critical: #f85149 (Red)
- Accent Warning: #d29922 (Amber)
- Text Primary: #c9d1d9
- Text Secondary (Muted): #8b949e
- Border: #30363d

## Typography
- Global Font: 'Courier New', monospace
- Card Headers: Bold, uppercase

## Components
- Cards: 1px solid #30363d border, 6px border-radius, background #161b22.
- Tabs: Transparent background, bottom-border highlights for active state.
- Status Indicator: Small glowing circle for 'Live' states.
- Code Blocks: Monospace text areas with distinct backgrounds for cryptographic data.
